# Muhammad Saqlain Ramzan - MERN Stack Portfolio

A professional, high-performance portfolio website built with the MERN stack (MongoDB, Express, React, Node.js) and styled with Tailwind CSS.

## Features
- **Modern UI/UX**: Responsive design with glassmorphism and smooth animations.
- **MERN Stack**: Fully functional contact form with data persistence in MongoDB.
- **Tailwind CSS**: Utility-first styling for maximum performance.
- **React Components**: Modular and reusable architecture.
- **Framer Motion**: Smooth entrance and scroll animations.
- **Lucide Icons**: Clean, consistent iconography.

## Prerequisites
- **Node.js** (v18 or higher)
- **MongoDB** (Local installation or Atlas URI)
- **npm** or **yarn**
